# LocalCut Pro 产品说明书

版本：V1.0  
日期：2026-06-04  
依据：基于当前代码库的产品与功能审核

## 1. 产品概述

LocalCut Pro 是一款运行在浏览器端的本地图片裁剪与区域导出工具。用户上传图片后，可以在画布中创建多个裁剪区域，支持矩形、圆形、三角形、菱形、星形、心形、自定义多边形和魔棒智能选区，并可通过画笔、橡皮和文字标注对图片进行轻量编辑。最终用户可将单个区域导出为图片，或将多个裁剪区域批量打包导出为 ZIP。

产品核心价值是让用户在不依赖后端、不上传原图的情况下，快速完成一张图片中多个素材区域的切割、命名、编辑与批量输出。

## 2. 目标用户

- 电商与运营人员：从商品图、活动图、物料图中快速切出多个可复用素材。
- 设计与内容制作人员：对图片局部区域进行快速标记、裁剪和导出。
- 游戏、教学、轻量素材整理场景用户：从大图、截图、素材图中提取多个不规则区域。
- 对隐私或本地处理有要求的用户：希望图片不离开本机浏览器环境。

## 3. 使用场景

- 从一张产品组合图中批量切出多个商品图。
- 对截图中的重点区域画框、加文字后导出局部图片。
- 使用魔棒从纯色或相近色背景中识别目标区域并导出。
- 用自定义多边形圈选不规则物体，生成透明边界裁剪结果。
- 对同一张原图创建多种形状裁剪框，并统一导出为 ZIP 文件。

## 4. 产品定位

LocalCut Pro 当前更接近“本地素材切割工作台”，不是完整的专业图片编辑器。它强调：

- 本地运行：图片处理、选区、导出均在浏览器前端完成。
- 多区域管理：同一图片可维护多个命名裁剪区域。
- 快速导出：支持单区域导出和多区域 ZIP 打包导出。
- 轻量编辑：提供基础画笔、橡皮和文字覆盖能力。
- 精确操作：提供画布缩放、平移、标尺、坐标与尺寸编辑。

## 6. 核心功能说明

### 6.1 图片导入

用户可通过左侧“上传图片”按钮选择本地图片，也可将图片拖拽到画布区域。加载新图片后，系统会清空已有裁剪区域、文字标注和多选状态，并基于原图尺寸初始化工作画布。

支持格式：

- PNG
- JPEG/JPG
- WebP

### 6.2 裁剪区域创建

用户可选择形状工具，在画布上拖拽创建区域。当前支持：

- 矩形
- 圆形
- 三角形
- 菱形
- 星形
- 心形
- 自定义多边形
- 魔棒智能选区生成的多边形

普通形状通过拖拽生成。自定义多边形通过连续点击添加顶点，支持点击起点、双击或按 Enter 完成闭合，按 Escape 取消当前绘制。

### 6.3 区域选择与编辑

用户可通过画布点击或右侧区域列表选择裁剪区域。选中区域后支持：

- 拖拽移动。
- 拖拽八个控制点调整尺寸。
- 在右侧面板精确编辑 X、Y、宽度、高度。
- 修改区域名称。
- 删除区域。
- 复制与粘贴区域。

自定义多边形额外支持：

- 拖拽顶点调整形状。
- 双击边线新增顶点。
- 右键点击顶点删除顶点。
- 顶点少于 3 个时区域自动失效。

### 6.4 魔棒智能选区

魔棒工具基于点击位置的像素颜色进行连通区域识别。用户可调整容差值，容差越高，被选中的相近颜色范围越大。

实现逻辑：

- 以点击像素为种子点。
- 对相近颜色区域执行连通填充。
- 提取区域轮廓。
- 简化轮廓点。
- 生成自定义多边形裁剪区域。

当前魔棒更适合背景清晰、色块边界较明确的图片；复杂纹理、渐变、阴影较多的图片可能需要手动微调多边形顶点。

### 6.5 画笔与橡皮

画笔用于直接在工作图层上绘制颜色，橡皮用于擦除工作图层内容。两者会影响后续导出的图片内容。

画笔设置：

- 笔刷大小：1 到 100 px。
- 笔刷颜色：任意颜色。

橡皮设置：

- 橡皮大小：1 到 150 px。

### 6.6 文字标注

用户选择文字工具后，可在画布上点击创建文字框。选中文字后，可在右侧面板编辑：

- 文字内容。
- 字号。
- 字重：常规或粗体。
- 颜色。
- 文字框位置与尺寸。

文字会参与区域导出。若文字框与裁剪区域重叠，导出时只保留裁剪区域内可见的文字部分。

### 6.7 画布查看

用户可通过滚轮缩放画布，缩放范围由系统限制，避免无限放大或缩小。按住 Space 并拖拽，或使用鼠标中键拖拽，可平移画布。

画布默认显示深色遮罩，裁剪区域以高亮轮廓显示。开启“原图”后，系统隐藏遮罩、区域覆盖和文字标注，便于检查原图内容。

### 6.8 导出

系统支持单区域导出和批量导出。

单区域导出：

- 对当前选中区域导出图片。
- 支持自定义输出尺寸。
- 支持设备像素比放大输出。
- 文件名使用区域名称。

批量导出：

- 默认导出全部区域。
- 若勾选了区域，则仅导出勾选区域。
- 输出为 ZIP 文件，默认文件名为 `output.zip`。
- 批量导出按裁剪区域原始宽高输出，不使用单区域的自定义输出尺寸。

导出格式：

- PNG
- JPEG
- WebP

JPEG 和 WebP 支持质量参数，范围为 10% 到 100%。PNG 不使用质量参数。

## 7. 快捷键

- Ctrl + Z：撤销。
- Ctrl + Y：重做。
- Ctrl + C：复制选中区域。
- Ctrl + V：粘贴区域。
- Delete / Backspace：删除选中区域。
- Space + 拖拽：平移画布。
- 鼠标滚轮：缩放画布。
- Enter：完成自定义多边形。
- Escape：取消正在绘制的自定义多边形。

## 8. 数据与隐私

LocalCut Pro 当前没有后端服务、账号体系或远程存储逻辑。图片读取、画布编辑、区域生成和文件导出均在浏览器本地完成。

当前版本不提供项目保存与恢复能力。刷新页面或关闭浏览器后，当前图片、裁剪区域、文字和编辑状态会丢失。

## 9. 当前版本范围与限制

本次审核发现，当前版本已具备完整的单图多区域裁剪导出闭环，但仍有以下边界需要在产品说明中明确：

- 撤销/重做当前主要覆盖裁剪区域，不完整覆盖文字标注、画笔和橡皮对工作图层的修改。
- 新图片加载后会清空原有区域与文字，不支持多图片项目管理。
- 未提供本地项目文件保存、打开、历史恢复能力。
- 魔棒识别依赖颜色相似度与连通性，对复杂图像结果可能不稳定。
- 批量导出固定使用区域原始尺寸，不应用右侧自定义输出尺寸。
- 区域名称要求非空且不可与其他区域重复，重复时会自动回退为原名称。
- 当前未看到移动端适配逻辑，产品主要面向桌面浏览器。
- 画笔/橡皮会直接修改工作图层，当前没有独立图层管理。

## 10. 非功能性要求

### 10.1 性能

- 应能流畅处理常见网页图片和电商图片。
- 对超大图片或复杂魔棒选区应给出合理性能边界。
- 魔棒连通填充已设置最大像素处理上限，避免极端情况下长时间阻塞。

### 10.2 兼容性

- 目标环境为现代桌面浏览器。
- 需要支持 Canvas、FileReader、Blob、ResizeObserver、download 等 Web 能力。

### 10.3 可用性

- 用户上传图片后应立即进入可编辑状态。
- 核心工具应通过图标、按钮状态和右侧属性面板提供明确反馈。
- 区域列表应持续展示当前区域数量和尺寸，便于批量管理。

### 10.4 安全与隐私

- 图片不上传服务器。
- 导出文件在本地生成并下载。
- 不采集用户账号、图片内容或编辑数据。

## 11. 验收标准

- 用户可以上传 PNG、JPEG、WebP 图片并在画布中正确显示。
- 用户可以创建所有预设形状的裁剪区域。
- 用户可以创建、完成、取消和编辑自定义多边形区域。
- 用户可以通过魔棒生成可编辑的多边形区域。
- 用户可以移动、缩放、重命名和删除裁剪区域。
- 用户可以添加文字，并调整内容、字号、粗细、颜色、位置和尺寸。
- 用户可以使用画笔和橡皮修改工作图层。
- 用户可以单独导出选中区域为 PNG、JPEG 或 WebP。
- 用户可以批量导出全部或勾选区域为 ZIP。
- 用户可以通过滚轮缩放、Space 拖拽平移画布。
- 生产构建应通过 `npm run build`。

## 12. 后续优化建议

- 增加项目保存与打开：将原图引用、区域、文字、导出设置保存为本地项目文件。
- 完善撤销/重做：覆盖文字、画笔、橡皮、导出设置和清空操作。
- 增加导出命名规则：支持前缀、序号、尺寸后缀、自定义 ZIP 名称。
- 增加批量导出尺寸策略：允许批量导出使用统一尺寸、倍率或最长边规则。
- 增加图层概念：将原图、画笔、文字、区域拆为更清晰的编辑层。
- 优化魔棒体验：加入选区预览、追加/减去选区、羽化或平滑参数。
- 增加对齐与参考线：支持吸附、等距复制、坐标辅助线。
- 增加键盘微调：方向键移动选中区域，Shift 加速移动。
- 增加空状态引导：中央画布可点击触发上传，减少用户误解。
- 增加移动端或窄屏降级方案，明确桌面端为优先体验。

## 13. 技术实现概览

当前项目基于 Vue 3、Vite、TypeScript 和 Pinia 实现。

- `src/App.vue`：应用布局、图片上传入口、全局快捷键。
- `src/components/TopBar.vue`：顶部品牌、设置、快捷键说明。
- `src/components/LeftSidebar.vue`：上传入口、工具列表、文字列表。
- `src/components/CanvasWorkspace.vue`：画布容器、标尺、拖拽上传、原图开关。
- `src/components/RightSidebar.vue`：属性编辑、导出设置、区域列表、批量导出。
- `src/composables/useCanvasEngine.ts`：画布渲染、交互、选区、绘制、缩放、平移。
- `src/composables/useExport.ts`：区域渲染、图片导出、ZIP 打包。
- `src/composables/magicWandUtils.ts`：魔棒智能选区算法。
- `src/composables/shapeUtils.ts`：裁剪形状路径绘制。
- `src/stores/editor.ts`：编辑器状态。
- `src/stores/history.ts`：裁剪区域历史记录。
- `src/stores/export.ts`：导出设置状态。

## 14. 审核结论

LocalCut Pro 当前已经形成一个清晰可用的 MVP：图片导入、多形状选区、智能选区、轻量编辑、属性管理和导出链路均已打通。产品适合作为本地图片多区域切割工具继续深化。

短期最值得补强的是“状态保存”和“完整撤销/重做”。这两项会直接影响用户处理复杂素材时的安全感和工作效率。其次是批量导出的命名与尺寸规则，这会提升产品在真实运营、设计交付场景中的可用性。
